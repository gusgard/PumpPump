# Installation

## Setup

Run `yarn install`

## Configure

Create a new file `api.config.js` based on `api.config.example.js`.
Add the following values:
* development
* production
* _version
* _SessionToken

## Start

Run `yarn start`

## Simulator

For an Android device `yarn android`

For an iOS device `yarn ios`
